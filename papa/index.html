PAPA MODE

History Crane Catcher

クレーンを左右に動かして景品をつかもう。ときどき出る歴史クイズに正解すると、時間やつかみやすさでボーナスがもらえる。

操作 ← / → と Space
目標 60秒でたくさん景品を取る
クイズ数 200問

ゲームの流れ

  1. 上下に動いて炎をよける
  2. クレーンを左右に動かす
  3. Spaceでアームを下ろす
  4. 景品が取れると得点アップ
  5. 歴史クイズでボーナスをねらう

ヒント

  • 景品の真上で止めるのがコツです
  • クイズ中はゲームが止まります
  • ベスト記録はブラウザに保存されます
