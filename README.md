# Penalty Kick Lab

ユーザーがキッカー、コンピューターがゴールキーパーを担当する、ブラウザ向けのPKゲームです。  
遊びながら、コース取り、威力調整、キーパーの癖の読み方を学べる構成にしています。

## できること

- 5本勝負のPKチャレンジ
- `←` `→` でコース選択、`↑` `↓` で威力調整、`Space` でシュート
- タイプ別に動くコンピューターキーパー
- 予想成功率とコースごとの特徴表示
- シュート結果ごとのコーチングコメント
- 試合ログと総合評価の表示

## 起動方法

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## 学べるポイント

- 高い隅は強力だが、威力が強すぎると枠外リスクが上がる
- 低い隅は安定しやすく、再現性の高い選択になりやすい
- キーパーの傾向を観察し、同じ高さや同じ側に偏らないことが重要
